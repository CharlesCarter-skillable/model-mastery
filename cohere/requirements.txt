# -----------------------------------------------------------------------------
# Microsoft Foundry / Azure SDKs
# -----------------------------------------------------------------------------
# `azure-ai-projects` is the Foundry SDK used throughout Lab 1 to talk to the
# project, list deployments, create/run prompt agents, drive the OpenAI-compatible
# chat client, and read evaluations. Pinned to v2.x preview because the prompt-agent
# `create_version` + `PromptAgentDefinition` / `FunctionTool` shapes used by
# `02-build-agent.ipynb` only exist in the v2 preview surface.
# `azure-ai-evaluation` exposes the built-in Quality / Safety / Agentic
# evaluators we use in Lab 1's three evaluation rounds, plus the base classes
# we extend for the custom policy-adherence evaluator.
# `azure-identity` provides DefaultAzureCredential (preferred over API keys).
# `azure-monitor-opentelemetry` wires traces/metrics to App Insights so the
# Foundry portal's Monitoring tab lights up during the Lab 1 load test.
